Cloud Fundamentals

Cloud Fundamentals is a very iffy course. On the one hand, I am now an AWS Cloud Practitioner. On the other hand, the class was incredibly memory based and I have absolutely no interest in pursuing cloud development.

AWS

AWS is very neat in and of itself. I love working with servers, and being able to reduce costs by aggregating usage over all users is immensely powerful for solo and indie developers. However, I just.. don't really care? The appeal to me, at least, is actually working with a physical system. With the cloud, that appeal is completely gone for me.

I may end up furthering this career path, but only to work with Amazon directly.
